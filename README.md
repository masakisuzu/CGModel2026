# CGモデル作品
TypeScriptで制作した3Dグラフィックスのシーン作品です

## 📕 タイトル
- 『夢の頂』
- アドベンチャーゲームのクライマックス等で訪問するような舞台設計を考えて制作しました。
- 宇宙まで届くとある施設の展望台で、流れ星が降り注ぐ舞台です。

## 📌 作品リンク
- [https://masakisuzu.github.io/my-browser-rpgcardgame/](https://masakisuzu.github.io/CGModel2026/)
- 公開URLにアクセスすることで閲覧することが出来ます
- クリックしながらカーソル移動することで視点変更が可能です
- マウスホイールで拡大縮小が可能です
